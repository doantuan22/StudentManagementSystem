# Database Scripts

- `01_create_database.sql`: khoi tao database `student_management`.
- `02_create_tables.sql`: tao day du cac bang va khoa ngoai chinh.
- `03_insert_sample_data.sql`: script nhanh nhat de nap role, user va du lieu mau test.
- `04_insert_roles.sql`: script tuong thich neu muon chen role rieng.
- `05_seed_sample_data.sql`: script tuong thich de nap du lieu mau sau khi co role.
- `06_create_views.sql`: cac view ho tro tra cuu va bao cao.
- `07_create_indexes.sql`: cac index toi uu truy van.
- `08_create_triggers.sql`: trigger muc khung.
- `09_create_procedures.sql`: stored procedure muc khung.
- `10_test_queries.sql`: truy van kiem tra du lieu.

## Thu tu khuyen nghi

Neu ban da tung tao `student_management` tu schema cu, hay reset truoc:

0. Chay `00_reset_database.sql`
1. Chay `01_create_database.sql`
2. Chay `02_create_tables.sql`
3. Chay `03_insert_sample_data.sql`
